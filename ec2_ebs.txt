What is EC2?
Elastic compute cloud is a service for creating and managing virtual machines in aws

for any ec2 resource - 
instance id - is the unique identifier for the instance
ARN - unique identifier in aws account
tags - Name: "value"

Default storage service for ec2 - EBS
Elastic Block storage

user data  - (bootstrap script)
we can run a bash script during the first startup/launch of the instance

create an instance 

in user data -

#!/bin/bash
mkdir /some
useradd testuser
chown -R testuser:testuser /some
echo "$(date) -- current time of execution" > /tmp/logfile
